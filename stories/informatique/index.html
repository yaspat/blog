Informatique

La procédure est la même, que vous soyez sur Windows ou Mac OS X.

1. Il ne faut pas installer Python tout seul, mais une distribution de Python : c'est-à-dire un ensemble d'applications et de services livrés avec Python. On travaillera avec la version 3.5 de Python.

  1. Vous aurez deux choses à installer :
  1. Pyzo : c'est l'environnement de développement disponible au lycée, et celle qui est utilisée au concours. C'est lui qui vous permet de dialoguer avec Python. Rendez-vous ici puis dans l'étape 1 (step 1: install the Pyzo IDE) choisissez le lien qui correspond à votre système. Puis, suivez l'étape 2 indiquée sur le site de Pyzo etc. Si les liens ne sont pas à jour, installez Anaconda comme indiqué au point ci-dessous.
  2. Anaconda : il permet une installation facile de Python, de ses packages, et est orienté calcul scientifique. Rendez-vous là et cliquez sur le logo correspondant à votre système (win, mac, ou linux). Choisissez l'installeur graphique de Python 3.6 (en vert, graphical installer). Lancez ensuite l'installation. Anaconda vous demandera si vous souhaitez recevoir leur anti-sèche (cheatsheet), vous pouvez décliner la proposition.
  3. Vous pouvez ensuite lancer Pyzo. Au démarrage, il vous devrait vous dire qu'il a détecté Anaconda et vous demandera si vous voulez en faire votre distribution par défaut. Sinon, il faut et lui faire savoir qu'il doit utiliser la distribution Python fournie par Anaconda de la manière suivante :
    • Allez dans le menu Shell
    • Sélectionnez Edit shell Configurations
    • Dans le champ exe, sélectionnez la distribtution Anaconda qui devrait apparaître.
    • Cochez la case ipython (Use IPython shell if avalaible).
    • Dans la fenêtre du shell (c'est-à-dire la console d'exécution de Python), lancez un nouveau shell pour que Pyzo ouvre une console exécutant la version de Python provenant de Anaconda.
  4. Happy pythoning !
